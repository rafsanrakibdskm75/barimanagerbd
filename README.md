# 🌟 About This Project

**Bari Manager BD (বাড়ি ম্যানেজার BD)** is a premium, full-featured property management and automated rent collection system designed specifically for building owners and caretakers in Bangladesh. It simplifies the tedious monthly tasks of tracking house inventories, managing tenant profiles, calculating electricity meter units, collecting utility bills, and auditing payments.

- ⚛ **React & TypeScript Frontend** for reliable runtime performance.
- ⚡ **Supabase Serverless Backend** handling authentication, REST APIs, and database migrations.
- 🎨 **MUI Glassmorphism Design System** featuring vibrant, high-fidelity widgets and dynamic charts.
- 🇧🇩 **Multilingual Interface** with rich Bengali language localization.
- 📱 **Fully Responsive Layout** optimizing views for desktops, tablets, and smartphones.

---

# 🛠 Tech Stack

## Frontend
- **Framework**: React 19 (SPA)
- **Programming Language**: TypeScript
- **UI Library**: Material UI (MUI v7)
- **Styling**: Emotion Styled Components
- **Data Visualization**: Recharts (Dynamic Income/Expense SVG charts)
- **Utilities**: `date-fns` for accurate period handling
- **Routing**: `react-router-dom` v7

## Backend & Database
- **Provider**: Supabase
- **Database**: PostgreSQL with Row-Level Security (RLS)
- **Data Migration**: Structured SQL migrations handling table cascading and constraints

# 📂 Features

✅ **Dashboard Analytics**: Beautiful graphic analytics illustrating monthly income, outstanding dues, paid vs. unpaid houses, and overall collections.  
✅ **Multi-House & Flat Manager**: Comprehensive register tracking floor count, room count, flat availability, and occupancy state.  
✅ **Tenant Profile Cards**: Complete profiles with full name, contact info, NID verification, move-in dates, emergency contacts, and active status.  
✅ **Utility Meter Calculator**: Standardized electricity billing interface that calculates units consumed (`current - previous reading`) and generates totals based on configured per-unit rates.  
✅ **Smart Rent Collection Ledger**: Auto-calculates totals including base rent, water bills, electricity dues, and service charges. Supports status flags like "পাওয়া গেছে" (Paid), "আংশিক" (Partial), and "বাকি আছে" (Pending).  
✅ **Collector & Ledger Audit**: Comprehensive payment logging history mapping the collector's name, phone number, transaction method (Cash, bKash, Nagad, Bank), and timestamps.  
✅ **Robust Backup Controls**: Export the entire database into a single JSON backup file and restore it with one click.  
✅ **Advanced Localization**: Deep integration of the Bengali language for intuitive real-world usage.
